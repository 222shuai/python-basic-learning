【软件开发】使用Python开发游戏-[飞机大战]

-------------没电脑的不要进来了---------
电脑上需要安装的三方库:
request,pygame
安装第三方库的方法:
举例我要安装pygame
同时按住键盘上win+r键
在弹出的左下角的框里输入 cmd
在跳出的黑色框里输入 pip install pygame
*注:如果要安装其他库，就把install后面的名字 改一下就可以
比如 pip install requests

安装requests的理由是，里面有自带的检测更新函数，不过是自行添加代码。

这个游戏没有使用精灵组，而是用列表代替，原因是我不怎么懂精灵组
